En este repositorio se muestra el diseño de una web para Pasteláctico, una empresa de venta de pasteles.
Se muestran tanto el logo de la empresa, así como la maquetación de dos páginas web;
La priemera para que el usuario haga sus pedidos, que para el caso tiene el nombre del neogcio.
La segunda para la gestión de la pastelería, cuyo nombre es ese.
